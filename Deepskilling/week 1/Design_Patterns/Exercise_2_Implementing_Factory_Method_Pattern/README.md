# Exercise 2: Implementing Factory Method Pattern

## Objective
This exercise demonstrates the Factory Method Design Pattern to define an interface for creating objects, but allowing subclasses to alter the type of objects that will be created (e.g. Word, Excel, and PDF Documents).

## Technologies Used
- Java

## How to Run
1. Navigate to this directory in your terminal.
2. Compile the Java files: `javac *.java`
3. Run the main test class to see object creation handled dynamically through the factory method.
