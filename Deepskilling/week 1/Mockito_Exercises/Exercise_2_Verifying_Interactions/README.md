# Exercise 2: Verifying Interactions

## Objective
This exercise demonstrates how to use Mockito to verify that specific methods on mock objects were called during the test execution, including the number of invocations and exact arguments.

## Technologies Used
- Java
- JUnit 5
- Mockito
- Maven

## How to Test
1. Navigate to this directory in your terminal.
2. Execute the tests via Maven:
   ```bash
   mvn test
   ```
