# Exercise 3: Stored Procedures

## Objective
This exercise covers how to create, execute, and manage Stored Procedures in PL/SQL to encapsulate business logic on the database side for improved performance and security.

## Technologies Used
- PL/SQL

## How to Run
1. Ensure you have an Oracle Database instance running.
2. Compile the stored procedures by executing the provided SQL scripts in your database client.
3. Call the procedures passing the required input/output parameters to observe the results.
