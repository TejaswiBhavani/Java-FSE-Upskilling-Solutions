# Exercise 1: Control Structures

## Objective
This exercise demonstrates the use of PL/SQL control structures (such as IF statements, loops, and cursors) to manage program flow logic directly within the database.

## Technologies Used
- PL/SQL

## How to Run
1. Ensure you have an Oracle Database instance or a compatible PL/SQL environment running.
2. Run the `SchemaSetup.sql` script first (if present).
3. Execute the Scenario scripts in your SQL client or CLI to test the control structures.
