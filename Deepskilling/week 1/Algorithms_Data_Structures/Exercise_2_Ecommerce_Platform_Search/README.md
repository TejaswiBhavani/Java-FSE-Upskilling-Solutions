# Exercise 2: Ecommerce Platform Search

## Objective
This exercise demonstrates algorithms for searching within an E-commerce platform, focusing on implementing and comparing linear search and binary search for product retrieval.

## Technologies Used
- Java

## How to Run
1. Navigate to this directory in your terminal.
2. Compile the Java files: `javac *.java`
3. Run the main class to see the search algorithms in action.
