# Exercise 2: Ecommerce Platform Search

## Objective
This exercise demonstrates algorithms for searching within an E-commerce platform, focusing on implementing and comparing linear search and binary search for product retrieval.

## Key Components
- **`EcommerceSearchTest.java`**: The main class that implements both the linear search and binary search algorithms. It contains the logic to create a mock array of products and test the execution time and correctness of each search approach.

## How to Test
1. Open a terminal and navigate to this directory.
2. Compile the Java file:
   ```bash
   javac EcommerceSearchTest.java
   ```
3. Run the compiled test class:
   ```bash
   java EcommerceSearchTest
   ```
