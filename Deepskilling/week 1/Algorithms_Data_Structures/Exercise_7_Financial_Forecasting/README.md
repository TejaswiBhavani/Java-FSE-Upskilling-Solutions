# Exercise 7: Financial Forecasting

## Objective
This exercise demonstrates the use of recursion to predict future financial values based on a growth rate over time.

## Key Components
- **`FinancialForecasting.java`**: The main class containing the recursive method to calculate the future value of a financial investment given a present value, growth rate, and number of periods. It also includes a `main` method to execute and test the calculation.

## How to Test
1. Open a terminal and navigate to this directory.
2. Compile the Java file:
   ```bash
   javac FinancialForecasting.java
   ```
3. Run the compiled test class:
   ```bash
   java FinancialForecasting
   ```
