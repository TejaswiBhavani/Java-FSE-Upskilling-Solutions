# Exercise 3: Assertions in JUnit

## Objective
This exercise demonstrates the use of standard JUnit assertion methods (such as `assertEquals`, `assertTrue`, `assertNotNull`, etc.) to effectively validate expected behaviors and outcomes in unit tests.

## Key Components
- **`pom.xml`**: Defines the Maven configuration with JUnit 5 dependencies.
- **`src/test/java/.../AssertionsTest.java`**: The core test class that showcases various JUnit assertions being used to validate different data types, conditions, and object states.

## How to Test
1. Open a terminal and navigate to this directory.
2. Run the tests using Maven:
   ```bash
   mvn test
   ```
