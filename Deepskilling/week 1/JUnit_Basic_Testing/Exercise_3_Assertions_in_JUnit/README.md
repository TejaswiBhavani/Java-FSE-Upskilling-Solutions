# Exercise 3: Assertions in JUnit

## Objective
This exercise demonstrates the use of standard JUnit assertion methods (such as `assertEquals`, `assertTrue`, `assertNotNull`, etc.) to effectively validate expected behaviors in unit tests.

## Technologies Used
- Java
- JUnit 5
- Maven

## How to Test
1. Navigate to this directory in your terminal.
2. Execute the tests via Maven:
   ```bash
   mvn test
   ```
