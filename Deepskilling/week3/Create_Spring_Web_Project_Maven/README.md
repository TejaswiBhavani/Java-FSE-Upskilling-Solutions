# Exercise 1 - Create Spring Web Project

## Objective
Create a Spring Boot Web project using Maven and understand the basic project structure.

## Technologies Used
- Java
- Spring Boot
- Maven
- Spring Web
- Spring Boot DevTools

## Project Structure
- src/main/java
- src/main/resources
- src/test/java
- pom.xml

## Steps Performed
1. Created the project using Spring Initializr.
2. Configured:
   - Group: com.cognizant
   - Artifact: spring-learn
3. Added dependencies:
   - Spring Web
   - Spring Boot DevTools
4. Built the project using Maven.
5. Imported the project into Eclipse.
6. Added logging to the `main()` method.
7. Successfully ran the application.
