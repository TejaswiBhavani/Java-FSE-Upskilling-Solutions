# Local Community Event Portal

This repository contains front-end exercises demonstrating HTML5, CSS3, JavaScript, and Bootstrap 5. All exercises are part of the "Local Community Event Portal" project.

## Structure
- \shared/\: Contains shared assets matching across modules.
- \HTML5/\: Semantic structures, forms, videos, and native APIs.
- \CSS3/\: Styling, pseudo-classes, responsive design, multi-column.
- \JavaScript/\: DOM manipulation, forms, fetch APIs, and JS fundamentals.
- \Bootstrap5/\: Responsive grid layouts, modern components, utilities.

## Setup
Simply clone this repository and open the specific \index.html\ files in your browser.
